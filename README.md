# Trabajo Práctico – Introducción a CI/CD con GitHub Actions

Objetivo general
El objetivo de este trabajo práctico es que los estudiantes comprendan el flujo básico de
integración continua (CI) utilizando herramientas reales de desarrollo. A lo largo de la
actividad deberán:
- Crear y configurar un repositorio en GitHub.
- Vincular Visual Studio Code con Git.
- Publicar un proyecto existente en un repositorio remoto.
- Comprender el flujo de commits y pushes.
- Configurar un workflow básico de GitHub Actions.
- Ejecutar automáticamente una verificación simple del proyecto.

La práctica está pensada como una primera aproximación al ecosistema CI/CD,
priorizando la comprensión conceptual del flujo de trabajo por sobre la complejidad
técnica.
Consigna general
Cada estudiante o grupo de estudiantes deberá tomar el proyecto base entregado por el
docente y construir un pipeline simple de integración continua utilizando GitHub
Actions.
El trabajo deberá demostrar:
- manejo básico de Git y GitHub,
- conexión entre entorno local y repositorio remoto,
- automatización de tareas simples mediante Actions,
- comprensión del concepto de integración continua.
